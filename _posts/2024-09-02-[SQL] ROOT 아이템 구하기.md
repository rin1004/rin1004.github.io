### 📜정답
-----
```
    SELECT A.ITEM_ID,A.ITEM_NAME 
    FROM ITEM_INFO AS A INNER JOIN ITEM_TREE AS B
    ON A.ITEM_ID=B.ITEM_ID
    WHERE B.PARENT_ITEM_ID IS NULL
```  
<br><br>

### 📜노트
-----
ROOT 아이템의 PARENT 아이템 ID는 NULL 이므로 PARENT_ITEM_ID가 NULL인 ITEM_ID를 가져오면 된다.
