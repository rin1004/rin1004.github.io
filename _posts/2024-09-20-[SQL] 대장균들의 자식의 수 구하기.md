### 📜정답
-----
```
    SELECT A.ID,COUNT(B.ID)AS CHILD_COUNT 
    FROM ECOLI_DATA  A LEFT JOIN ECOLI_DATA B 
    ON A.ID=B.PARENT_ID
    GROUP BY ID
    ORDER BY ID ASC
```
  
    
### 📜노트
-----
`A테이블`은 `PARENT`,`B테이블`은 `CHILD`로 볼 수 있다.  
`PARENT`를 기준으로 `ARENT_ID`와 `ID`를 `LEFT JOIN`하는 방식
